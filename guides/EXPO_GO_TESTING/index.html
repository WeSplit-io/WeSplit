    How to Test Properly¶

    Option 1: Development Build (Recommended)¶

    Create a development build that includes native modules:

    # Install EAS CLI
npm install -g eas-cli

# Configure EAS
eas build:configure

# Build development version
eas build --profile development --platform ios
# or
eas build --profile development --platform android

    Benefits: - ‚úÖ Native modules work (Keychain/MMKV) - ‚úÖ Tests actual production behavior - ‚úÖ Can test on physical device - ‚úÖ Same code as production

    Limitations: - ‚ö†Ô∏è Requires EAS account - ‚ö†Ô∏è Takes time to build

    Option 2: Local Development Build¶

    Build locally (faster, but requires setup):

    # iOS

# Android
npx expo run:android

    Benefits: - ‚úÖ Native modules work - ‚úÖ Tests actual behavior - ‚úÖ Fast iteration

    Limitations: - ‚ö†Ô∏è Requires Xcode (iOS) or Android Studio (Android) - ‚ö†Ô∏è Mac required for iOS

    Build production version and test via TestFlight:

    Limitations: - ‚ö†Ô∏è Requires Apple Developer account - ‚ö†Ô∏è Takes time to build and distribute
